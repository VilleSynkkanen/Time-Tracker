Key features (done):
-Tracker app
	-Tracking statistics
		-application name
		-tracking started
		-use time
		-last used
-GUI (using PyQt)
	-Edit application name
	-Favorites, hidden programs
	-Sorting
	-Launching GUI stops tracker
	-Closing starts tracker again

Problems:
-saving when tracking process is killed does not work due to Windows not supporting signals properly
	-means that last 5 minutes of use are lost

How to run:
-run the included bat file

Run on startup:
-add a shortcut to the bat file to the startup folder of start menu 
(C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup)

Dependencies:
-WMI
-psutil
-pyqt5
-jsons
-win32process
